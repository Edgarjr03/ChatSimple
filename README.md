# ChatSimple / Simple Chat

## Español
Aplicación de chat en tiempo real desarrollada en C#.
Permite la comunicación entre un cliente y un servidor mediante sockets TCP/IP,
configurando la dirección IP y el puerto manualmente.

### Tecnologías
- C#
- Sockets TCP/IP
- Windows Forms
- Visual Studio 2022

### Cómo ejecutar
1. Clona el repositorio
2. Abre el archivo `.sln` en Visual Studio
3. Ejecuta primero el servidor
4. Ejecuta el cliente e ingresa la IP y puerto del servidor
5. ¡Empieza a chatear!

---

## English
Real-time chat application developed in C#.
Allows communication between a client and a server using TCP/IP sockets,
with manually configurable IP address and port.

### Technologies
- C#
- TCP/IP Sockets
- Windows Forms
- Visual Studio 2022

### How to run
1. Clone the repository
2. Open the `.sln` file in Visual Studio
3. Run the server first
4. Run the client and enter the server's IP and port
5. Start chatting!
